# OpenCart End-to-End Testing

## Overview

This project is an end-to-end automation testing framework developed for the OpenCart e-commerce application. The application is deployed locally using XAMPP, with Apache serving the web application and MySQL managing the database. Automated test cases are implemented using Selenium WebDriver with Java and executed through the TestNG framework.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* MySQL
* XAMPP (Apache & MySQL)
* Extent Reports
* Maven
* TestNG XML

## Features

* End-to-end automation testing of OpenCart workflows
* Data-driven and functional test execution
* Database validation using MySQL
* Parallel test execution through TestNG XML configuration
* Test lifecycle management using TestNG annotations
* Detailed execution reports generated with Extent Reports
* Reusable and maintainable automation framework structure

## Test Scenarios Covered

* User Registration
* User Login and Logout
* Product Search
* Product Selection
* Add to Cart
* Checkout Process
* Order Validation
* Account Management
* Database Verification

## Project Structure

* Test Scripts implemented using Selenium WebDriver
* TestNG framework for execution and test management
* XML configuration for suite execution and parallel testing
* Extent Reports for test result reporting
* MySQL database validation for backend verification

## How to Run

1. Start Apache and MySQL services using XAMPP.
2. Deploy and launch the OpenCart application locally.
3. Configure database and application settings.
4. Execute the TestNG XML suite file.
5. View execution results in the generated Extent Report.

## Objective

The objective of this project is to demonstrate practical implementation of web automation testing, database testing, parallel execution, reporting, and framework design using industry-standard tools and best practices.
